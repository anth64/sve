# sve (Stalwart Voxel Engine)

[![License: MPL 2.0](https://img.shields.io/badge/License-MPL_2.0-brightgreen.svg)](https://opensource.org/licenses/MPL-2.0)

`sve` is a lightweight, portable voxel engine written in C99, inspired by 90s rendering techniques. It favours software rendering over hardware, running on any hardware without a GPU requirement.

## Status

Early development. Nothing works yet.
